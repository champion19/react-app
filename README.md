# 2. iniciar la app de react

- Instalar [node.js](https://nodejs.org/en/download/current/)
- Instalar [yarn](https://classic.yarnpkg.com/lang/en/docs/install/#windows-stable)
- Ejecutar npm -v ó yarn -v
- ``npm install`
- ``cd react-app`
- `npm start`
